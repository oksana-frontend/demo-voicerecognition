HTML5 demo страница полезной информации для выходных в Киеве. 


API

Содержит 3 функции
function isRecognitionAvailable() - проверяет доступна ли голосовая 
функция в браузере
function startDictation(onResult, onError) - начинает распознавание
принимает две функции - onResult (вызывается, когда есть результат 
распознавания) onError (когда ошибка распознавания)
function stopDictation() - остановка распознавания




Голосовой поиск работает со следующими командами: (переходит к нужному
разделу)

такси
достопримечательности
рестораны
кофейни
магазины
карта метро
программа


Для того, чтобы голосовой поиск начал работать нужно нажать микрофон и 
начать говорить.
Для завершения распознавания нужно снова нажать микрофон.
